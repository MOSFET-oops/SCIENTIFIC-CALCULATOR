Scientific Calculator

A command-line scientific calculator developed in C.

>> Features
- Basic arithmetic operations
- Trigonometric functions
- Logarithmic functions
- Exponential and power calculations
- Square root
- Factorial
- User-friendly menu interface

 >> Technologies Used
- C Programming
- GCC / MinGW-w64
- Git & GitHub

>> Purpose
This project was created to strengthen my understanding of C programming, functions, mathematical libraries, and version control using Git and GitHub.
