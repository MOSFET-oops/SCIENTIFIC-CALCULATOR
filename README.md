# 🧮 Scientific Calculator in C

A feature-rich **console-based Scientific Calculator** developed in **C** using the standard `math.h` library. This project was built to strengthen core programming concepts including functions, switch-case statements, modular programming, mathematical computations, and user input handling.

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Scientific-Calculator-in-C.git
```

### Compile

```bash
gcc Scientific_Calculator.c -o calculator.exe -lm
```

### Run

**Windows (PowerShell)**

```powershell
.\calculator.exe
```

**Linux / macOS**

```bash
./calculator
```
